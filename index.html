Census 2011 India Data Analysis

This dataset has been created by merging the census 2011 of Indian Cities with Population more than 1 Lac and City wise number of Graduates from the Census 2011. The Census is one of the biggest workout conducted by the Government of India to understand the demographics of the country. It is conducted every 10 years. The Britishers initiated it during the colonial times. It helps in a big way to come up with social and economic programs in the country. It also helps to identify the problems in the country. We can use this dataset to create a visualization to see where the future cities of India stand today. More information can be found

http://www.census2011.co.in

About the Visualization

This visualization project is following the Martini Glass narrative visualization structure. Here first page provides the details about the dataset and visualization. The second and third page provide the overview of Adult and Child population of various states in the country.

The first three scenes provides the author driven content.

Analysis page contains reader driven content and it allows users to interact with data and draw their own conclusions.

All the chart are built using d3.js

User Interface Events

The navigation buttons at the top and the Next button at the bottom of all page represent the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this visualzation story.

Annotations

The annotation is represented by the legend on the top-right of the chart.

Triggers

The mouse click event acts as a trigger to faciliate back and forward navigation in this visualization presentation.
