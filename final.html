The project is following the Martini Glass narrative visualization structure. This page represents the final chapter of this visualization structure. The users is ready to explore reader driven contents and draw own conclusions.

User Interface Events

The buttons in the navigation bar and the Prev, Next button at the bottom of this page enable the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this story.

User is presented with helpful hints as the part of page load event. The alert can be closed by clicking on X or it will automatically disappear after six seconds, allowing enough time for user to read the information.

The mouse hover event on the either map shows the tooltip containing additional informaton about that specific page.

The mouse click event on the either map causes the bar chart in the bottom half to automatically refresh and show data for the state clicked in the map.

Annotations

Both maps uses tool tips and title for annotation. The bar charts uses legend and numerical value in the individual map for annotation.

Triggers

The mouse click and mouse hover events are the triggers for this page.

Parameters

The backend d3 javascript for the map passes the state information and the div information to the javascript function rendering the bar charts. On page load, the bar chart show the data of California. If user clicks on a state in map, it receives divID, and state id as input paramter and refreshes bar chart with the data for the requested state.
